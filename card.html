Hello Friend (WIP)

I created this digital card maker for a p5.js class. Currently, you can type in a greeting and click on the card to decorate it. Each letter is hand-lettered. This is a work in progress as there are still some areas that do not work well (using keys that are not letters) and I am working to add more functionality (adding SPACE, ENTER, BACKSPACE). My goal is to add a save canvas function so that the card can be saved as a GIF.

Tools

Sublime Text, p5 editor, pencil for lettering, Photoshop

Process

1 week to code and draw letters
